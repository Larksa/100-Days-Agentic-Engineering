# 100 Days of Agentic Engineering

**Learn to code by building 100 real projects with an AI partner**

---

This is a hands-on coding course where you and an AI partner build 100 real projects together, one per day. You will go from zero experience to writing real software -- apps, tools, automations, and more. Unlike traditional courses, you are not watching videos or reading textbooks. You are building from day one, with an AI guide that adapts to your pace and answers your questions as you go.

## What You Need

- A computer (Mac, Windows, or Linux)
- An internet connection

**Note:** Claude Code requires a Claude subscription (Pro, Team, or Enterprise). See <https://claude.ai/code> for details and pricing.

## Getting Started

**Step 1: Install Claude Code**

Visit <https://claude.ai/code> and follow the instructions to install.

- **Mac:** Follow the installer.
- **Windows:** Install WSL first by following the guide at <https://learn.microsoft.com/en-us/windows/wsl/install>, then install Claude Code inside WSL.
- **Linux:** Follow the installer.

**Step 2: Open Your Terminal**

- **Mac:** Press Command + Space to open Spotlight, then type "Terminal" and press Enter.
- **Windows:** After installing WSL, open the Start menu and search for "Ubuntu".
- **Linux:** Open your terminal application.

**Step 3: Navigate to This Folder and Start**

Type the following commands, pressing Enter after each one:

```
cd 100-Days-Agentic-Engineering
```

```
claude
```

That's it -- Claude will guide you from here.

## Platform Notes

This course works on Mac, Windows (via WSL), and Linux. All instructions inside the course are given to you by Claude Code directly, so you do not need any other materials.

## Frequently Asked Questions

**What language will I learn?**

Python by default, but you can ask Claude to teach in JavaScript, TypeScript, Java, or any language you prefer. Just say so when you start.

**How long does each day take?**

Most days take between 45 and 90 minutes. There is no timer and no deadline. Go at your own pace.

**Do I need to do them in order?**

Recommended but not required. Each day builds on concepts from previous days, so following the order will give you the smoothest experience. That said, if a particular project catches your eye, feel free to jump ahead.
